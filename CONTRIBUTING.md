# Contributing to Arabic AI Toolkit

We love your input! We want to make contributing to this project as easy and transparent as possible, whether it's:
- Reporting a bug
- Discussing the current state of the code
- Submitting a fix
- Proposing new features

## Development Process
1. Fork the repo and create your branch from `main`.
2. Install dependencies: `pip install -e .[dev]`
3. Run tests before submitting: `pytest`
4. Ensure your code is formatted and passes linting: `ruff check .`
5. Ensure type checking passes: `mypy src/`

## Pull Request Process
1. Update the README.md with details of changes to the interface, if applicable.
2. Update the CHANGELOG.md.
3. Your PR will be merged once you have the sign-off of at least one maintainer.
